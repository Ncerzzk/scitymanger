## 基于超图的城市可视化示例

### 使用

#####  打开 https://gitee.com/ 登录

#####  新建仓库 选择导入仓库 https://github.com/zhangti0708/cesium-city3d.git

#####  创建仓库并 git clone 到本地即可. 一分钟左右

#####  unzip cesium-city3d.zip

#####  将cesium-city3d目录直接放在web服务器下直接浏览
